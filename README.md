> A Vue.js project

## Commands
* dev: `yarn dev`
* build: `yarn build`

## Technologies
This uses Nuxt.js to build a static website for me based on contentful entries. It is set to auto deploy from Netlify when it is updated via a repository push or when contentful is updated.
